# Rilevamento di frodi nelle transazioni con carta di credito
## Introduzione
In questo progetto analizzo un dataset di transazioni con carta di credito con l'obiettivo di individuare possibili frodi.
Il problema e interessante perche le frodi sono poche rispetto alle transazioni normali. Questo rende il dataset molto sbilanciato e rende poco utile guardare solo l'accuracy.
## Obiettivo del progetto
L'obiettivo e costruire un percorso semplice e comprensibile:
- caricare il dataset in modo automatico;
- descrivere i dati principali;
- controllare lo sbilanciamento della variabile target;
- provare alcuni metodi di anomaly detection;
- addestrare un modello supervisionato;
- confrontare i risultati con metriche adatte al problema.
